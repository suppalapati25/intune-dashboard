Intune Device Dashboard
Enterprise device management & update intelligence
Sign in with your Microsoft 365 admin account to load real data
1
Create an App Registration (one-time, 2 min)
Go to portal.azure.com → Entra ID → App registrations → New registration
• Name: Intune Dashboard
• Supported account types: Single tenant
• Redirect URI: Single-page application (SPA) → enter the URL of this file
  (e.g. http://localhost:3000 or your hosted URL)
• Click Register → copy the Application (client) ID
2
Add API Permissions
In the app → API permissions → Add permission → Microsoft Graph → Delegated
Add: DeviceManagementManagedDevices.Read.All and DeviceManagementConfiguration.Read.All
Then click Grant admin consent
3
Paste your Client ID & Tenant ID below
Client ID is on the app overview page. Tenant ID is in Entra ID → Overview
